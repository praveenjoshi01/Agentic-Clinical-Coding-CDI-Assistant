Create all Pydantic v2 BaseModel schemas that define inter-module contracts. These are the single source of truth for the entire pipeline.

**cliniq/models/document.py:**
- `DocumentMetadata`: patient_id (str), encounter_id (str), source_type (Literal["fhir", "image", "text"]), timestamp (datetime, default=now)
- `ClinicalDocument`: metadata (DocumentMetadata), raw_narrative (str), structured_facts (list[dict], default_factory=list), modality_confidence (float, ge=0.0, le=1.0), extraction_trace (str, default="")

**cliniq/models/entities.py:**
- `ENTITY_TYPE_MAP`: dict mapping d4data model labels to pipeline categories. Keys: Disease_disorder, Sign_symptom -> "diagnosis"; Therapeutic_procedure, Diagnostic_procedure -> "procedure"; Medication -> "medication"; Biological_structure -> "anatomical_site"; Severity, Detailed_description, Qualitative_concept, Quantitative_concept -> "qualifier"; Lab_value -> "lab_value". Store as module-level constant.
- `ClinicalEntity`: text (str), entity_type (str), start_char (int), end_char (int), confidence (float, ge=0.0, le=1.0), negated (bool, default=False), qualifiers (list[str], default_factory=list)
- `NLUResult`: entities (list[ClinicalEntity]), processing_time_ms (float). Add computed properties: diagnoses (filter entity_type=="diagnosis"), procedures (filter entity_type=="procedure"), medications (filter entity_type=="medication"), anatomical_sites (filter entity_type=="anatomical_site"). Add entity_count property.

**cliniq/models/coding.py:**
- `CodeSuggestion`: icd10_code (str), description (str), confidence (float, ge=0.0, le=1.0), evidence_text (str), reasoning (str), needs_specificity (bool, default=False), alternatives (list[dict], default_factory=list)
- `CodingResult`: principal_diagnosis (CodeSuggestion), secondary_codes (list[CodeSuggestion], default_factory=list), complication_codes (list[CodeSuggestion], default_factory=list), sequencing_rationale (str, default=""), retrieval_stats (dict, default_factory=dict)

**cliniq/models/evaluation.py:**
- `GoldStandardEntity`: text (str), entity_type (str), start_char (int), end_char (int), negated (bool), qualifiers (list[str], default_factory=list)
- `GoldStandardCase`: case_id (str), source_type (Literal["fhir", "text", "image"]), input_data (str -- file path or inline text), expected_entities (list[GoldStandardEntity]), expected_icd10_codes (list[str]), expected_principal_dx (str), expected_comorbidities (list[str], default_factory=list), expected_complications (list[str], default_factory=list), negation_test_cases (list[dict], default_factory=list), cdi_gap_annotations (Optional[list[dict]], default=None), kg_qualification_rules (Optional[list[dict]], default=None), notes (str, default="")
- `EvalMetrics`: dict[str, float] wrapper or just use dict
- `EvalResult`: module_name (str), timestamp (datetime), n_cases (int), metrics (dict[str, float]), per_case_results (list[dict], default_factory=list), failures (list[dict], default_factory=list), passed (bool)

**cliniq/models/__init__.py:** Re-export all public classes: ClinicalDocument, DocumentMetadata, ClinicalEntity, NLUResult, CodeSuggestion, CodingResult, GoldStandardCase, GoldStandardEntity, EvalResult, ENTITY_TYPE_MAP.
