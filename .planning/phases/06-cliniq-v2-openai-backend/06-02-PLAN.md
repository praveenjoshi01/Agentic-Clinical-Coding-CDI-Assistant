**cliniq_v2/rag/build_index.py:**
- Import load_icd10_codes from cliniq.rag.icd10_loader (reuse).
- Import INDEX_DIR, EMBEDDING_DIMENSIONS from cliniq_v2.config.
- Import OpenAIClient from cliniq_v2.api_client.
- Write `build_faiss_index(codes=None, output_dir=None)`:
  1. Load codes via load_icd10_codes() if not provided
  2. Use output_dir or INDEX_DIR (cliniq_v2 cache, not cliniq cache)
  3. Create output_dir if not exists
  4. Extract descriptions from codes
  5. Call OpenAI Embeddings API in batches of 2048:
     `client.embeddings.create(model="text-embedding-3-small", input=batch)`
  6. Collect all embeddings into numpy array (dtype=float32), shape (N, 1536)
  7. Create faiss.IndexFlatIP(1536), add embeddings
  8. Save index to "icd10.faiss" and metadata to "icd10_metadata.json"
  9. Return (index, codes)
- Write `load_faiss_index(index_dir=None)`:
  Same as cliniq version but defaults to cliniq_v2 INDEX_DIR.

**cliniq_v2/rag/retriever.py:**
- Import INDEX_DIR from cliniq_v2.config.
- Import OpenAIClient from cliniq_v2.api_client.
- Import build_faiss_index, load_faiss_index from cliniq_v2.rag.build_index.
- Write `FAISSRetriever` class:
  - `__init__(index_dir=None)`: uses cliniq_v2 INDEX_DIR as default
  - `_ensure_loaded()`: lazy loads index, does NOT load local embedder
  - `ensure_index_built()`: checks for index, calls build_faiss_index if missing
  - `retrieve(query, top_k=RETRIEVAL_TOP_K)`:
    1. Call _ensure_loaded()
    2. Encode query via `OpenAIClient().client.embeddings.create(model="text-embedding-3-small", input=query)`
    3. Extract embedding, reshape to (1, 1536), dtype float32
    4. Search FAISS index
    5. Return list of dicts with code, description, score, rank
  - No BGE_QUERY_PREFIX needed (OpenAI embeddings don't use query prefixes)
